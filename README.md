# 四时有序 · 六爻解卦

一款六爻纳甲解卦工具，现已升级为前后端分离架构：

- 前端负责排盘解析、规则引擎、可视化展示
- 后端负责 AI 代理、配额管理、归档接口

## 在线使用

**https://junxuezha1.github.io/do-not-behind/**

手机浏览器打开后，点击「添加到主屏幕」可像 APP 一样使用。

## 功能特性

- **卦盘解析** — 粘贴灵光象吉等排盘软件的输出文本，自动识别解析
- **规则引擎** — 自动分析空亡、月建、日辰、动变、旺衰、进退神等
- **AI 研判** — 通过后端统一代理 DeepSeek，前端无需配置 API Key
- **配额控制** — 后端按会话/IP 管控每日免费次数
- **应期推断** — 基于空亡、动变、合冲等规则推算应期线索
- **归档系统** — 优先存入后端归档接口，后端不可用时本地兜底
- **干支日历** — 显示每日天干地支与旬空，方便择日参考
- **五行配色** — 爻线、六神、地支等均按五行着色（木绿火红土黄金白水蓝）
- **PWA 支持** — 可离线使用，支持安装到手机主屏幕

## 使用方法

1. 从灵光象吉等排盘软件复制卦盘文本
2. 粘贴到输入框，点击「解析卦盘」
3. 查看基本信息、规则分析、应期等标签页
4. 切换到「AI 研判」标签页，直接点击「AI 断卦」
5. 点击「归档此卦」保存分析结果

## AI 服务配置

AI Key 只放在后端环境变量，不会暴露给浏览器。

- 后端环境变量：`DEEPSEEK_API_KEY`
- 默认模型：`deepseek-chat`
- 默认端口：`8081`

## 技术栈

前端：

- HTML + CSS + JavaScript
- Service Worker 离线缓存

后端：

- Node.js + Express
- SSE 转发 DeepSeek 流式输出
- JSON 文件存储归档与配额日志

## 本地开发

1. 启动前端静态服务（项目根目录）

```bash
python -m http.server 8080
```

2. 启动后端（`backend/` 目录）

```bash
cp .env.example .env
# 填入 DEEPSEEK_API_KEY
npm install
npm start
```

3. 浏览器访问

```text
http://localhost:8080
```
